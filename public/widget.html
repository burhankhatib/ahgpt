How to Integrate

Integrating the Al Hayat GPT widget into your website is simple. Just include our JavaScript SDK and create a widget instance with your configuration: 

<!-- Include the Al Hayat GPT Widget SDK -->
<script src="https://www.alhayatgpt.com/widget-sdk.min.js"></script>

<!-- Create a container for the widget -->
<div id="chat-widget"></div>

<script>
// Initialize the widget
AlHayatGPT.createWidget({
    containerId: 'chat-widget',
    apiEndpoint: 'https://www.alhayatgpt.com',
    allowGuests: true
});
</script>
